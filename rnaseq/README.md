# Docker image for RNA-Seq processes

It is based on `ubuntu-16.04` version of [`docker.io/resolwe/base`](
https://hub.docker.com/r/resolwe/base/) image.

Included bioinformatics tools:
------------------------------
* cutadapt (v1.9.1)
* FastQC (v0.11.4)
* HTSeq (v0.5.4)
* SAMtools (v0.1.19)
* STAR (v2.5.3a)
